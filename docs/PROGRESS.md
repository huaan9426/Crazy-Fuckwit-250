`src/` 是前端体验主线，当前负责可玩性、视觉表现、交互节奏和本地兜底。`server/` 是 Go 后端主线，当前负责 API 形状、内容包、用户名占用、成绩提交和排行榜。两者是同一个项目的两个部分，不应该把其中任何一个单独当成完整主工程。
